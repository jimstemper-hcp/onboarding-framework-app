# Weekly Planning

## Overview
The Weekly Planning view organizes onboarding items into a 4-week structure, allowing reps to create a structured, time-based onboarding plan for each pro.

## Key Features
- 4-week grid layout
- Drag-and-drop item assignment
- Week progress tracking
- Item reordering within weeks
- Unassigned items pool

## User Stories
- As a rep, I want to assign items to specific weeks so pros have clear weekly goals
- As a rep, I want to reorder items within a week so priority is clear
- As a rep, I want to see week progress so I can track pacing
- As a rep, I want to move items between weeks as plans change

## UI Components
- Week column cards (Week 1-4)
- Draggable item cards
- Week progress header
- Unassigned items section
- Drop zones for drag-and-drop

## Data Dependencies
- Reads: Onboarding items, weekly plan assignments
- Writes: Weekly plan item assignments, item order

## Status
