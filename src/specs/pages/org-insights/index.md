# Org Insights Admin Panel

## Overview
The Org Insights Admin Panel is the rep-facing interface for managing individual pro onboarding. Reps use this panel to monitor progress, customize onboarding plans, schedule calls, and view feature adoption status.

## Key Features
- Pro account selection and overview
- Information dashboard
- Onboarding plan management (category and weekly views)
- Feature adoption status list
- Calendly call scheduling

## User Stories
- As a rep, I want to see all my assigned pros so I can manage their onboarding
- As a rep, I want to view a pro's progress so I can identify areas needing help
- As a rep, I want to customize onboarding plans so they match pro goals
- As a rep, I want to schedule calls easily so I can provide timely support

## UI Components
- Sidebar navigation with page tabs
- Pro selector dropdown
- Page-specific content areas
- Data tables and cards

## Data Dependencies
- Reads: Pro accounts, features, onboarding items, calendly links
- Writes: Onboarding plan assignments, task status

## Status
Prototype - Core admin panel functionality for rep use.
