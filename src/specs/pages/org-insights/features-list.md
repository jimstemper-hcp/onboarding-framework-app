# Features List

## Overview
The Features List page displays all features available to the selected pro along with their current adoption stage. This helps reps understand which features the pro is using and where they are in the adoption lifecycle.

## Key Features
- Feature list with adoption stages
- Stage indicators (Not Attached, Attached, Activated, Engaged)
- Feature icons and descriptions
- Usage count display
- Stage change capabilities

## User Stories
- As a rep, I want to see all features for a pro so I understand their full potential
- As a rep, I want to see adoption stages so I can focus on moving pros forward
- As a rep, I want to see usage counts so I understand engagement levels
- As a rep, I want to help pros advance to higher adoption stages

## UI Components
- Feature cards with icons
- Stage badge chips (color-coded)
- Usage statistics
- Progress indicators
- Feature detail expansion

## Data Dependencies
- Reads: Features, pro's feature status, usage counts
- Writes: Feature stage updates (potentially)

## Status
Prototype - Feature adoption visibility for reps.
