# Onboarding Plan

## Overview
The Onboarding Plan page allows reps to view and manage the selected pro's onboarding plan. It supports two viewing modes: category-based and weekly-based, giving reps flexibility in how they organize and track onboarding items.

## Key Features
- Category view: Items grouped by onboarding category
- Weekly view: Items organized by week (1-4)
- Item completion tracking
- Drag-and-drop reordering (weekly view)
- Add/remove items from plan

## User Stories
- As a rep, I want to see items by category so I can understand coverage areas
- As a rep, I want to see items by week so I can plan sequential tasks
- As a rep, I want to customize the plan so it fits the pro's specific needs
- As a rep, I want to track completion so I know what's been done

## UI Components
- View mode toggle (Category/Weekly)
- Category accordions with item lists
- Weekly cards with task items
- Item search and filter
- Completion checkboxes

## Data Dependencies
- Reads: Onboarding items, categories, pro's plan assignments
- Writes: Plan assignments, item order

## Status
Prototype - Dual-view plan management interface.
