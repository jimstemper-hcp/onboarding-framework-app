# Calls Page

## Overview
The Calls Page provides reps with easy access to schedule calls with the selected pro using Calendly. It displays available call types and their booking links organized by team.

## Key Features
- Calendly link integration
- Call types organized by team
- Direct booking buttons
- Call duration and description display
- Scheduling status

## User Stories
- As a rep, I want to schedule calls quickly so I can help pros efficiently
- As a rep, I want to see available call types so I can choose the right one
- As a rep, I want team-organized links so I can route to the right specialist
- As a rep, I want to see call durations so I can plan my schedule

## UI Components
- Team-grouped call type cards
- Booking buttons with Calendly links
- Duration and description text
- Status indicators

## Data Dependencies
- Reads: Calendly links, call types, team assignments
- Writes: None (redirects to Calendly)

## Status
Prototype - Calendly integration for call scheduling.
