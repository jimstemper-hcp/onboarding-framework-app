# Housecall Pro Web

## Overview
The pro-facing web portal that serves as the primary onboarding interface for Housecall Pro service professionals. This portal provides a gamified onboarding experience with two distinct viewing modes.

## Key Features
- Journey View: Timeline-based gamified onboarding experience
- Weekly View: Task-focused weekly planning interface
- Progress tracking with points and streaks
- Feature milestone tracking
- Personalized onboarding plans based on pro goals

## User Stories
- As a pro, I want to see my onboarding progress visually so that I stay motivated
- As a pro, I want to complete tasks at my own pace so that I can manage my time effectively
- As a pro, I want to earn points for completing tasks so that I feel rewarded

## UI Components
- Progress summary card with points, percentage, and streak counter
- View mode toggle (Journey/Weekly)
- Feature timeline with expandable milestones
- Task completion checkboxes

## Data Dependencies
- Reads: Pro account data, feature definitions, onboarding items
- Writes: Task completion status, progress tracking

## Status
Prototype stage - Demonstrates the core onboarding portal experience.
