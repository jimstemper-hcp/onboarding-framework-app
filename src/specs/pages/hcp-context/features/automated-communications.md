# Automated Communications Feature

## Overview
The Automated Communications feature enables pros to set up automated messages to customers for appointment reminders, follow-ups, and review requests.

## Key Features
- Appointment reminders (SMS/Email)
- On-my-way notifications
- Follow-up messages
- Review request automation
- Custom message templates

## Stage Contexts

### Not Attached
- Benefit messaging: "Save time with automated customer messaging"
- Communication efficiency stats

### Attached
- Onboarding items: Enable appointment reminders, customize templates
- Quick setup guides

### Activated
- Onboarding items: Set up review requests, configure follow-ups
- Advanced automation features

### Engaged
- Message performance analytics
- A/B testing guidance

## Data Dependencies
- Reads: Appointment data, customer data, message templates
- Writes: Message settings, template customizations

## Status
Shipped - Communication automation available to all pros.
