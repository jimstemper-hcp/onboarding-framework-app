# Reporting Feature

## Overview
The Reporting feature provides pros with business analytics and insights through customizable reports and dashboards.

## Key Features
- Pre-built report templates
- Custom report builder
- Dashboard widgets
- Export capabilities
- Scheduled report delivery

## Stage Contexts

### Not Attached
- Benefit messaging: "Make data-driven decisions with reports"
- Business insight benefits

### Attached
- Onboarding items: View first report, explore dashboard
- Report basics

### Activated
- Onboarding items: Customize dashboard, create custom report
- Advanced reporting

### Engaged
- Analytics mastery
- Data-driven strategies

## Data Dependencies
- Reads: All business data (jobs, customers, payments, etc.)
- Writes: Report configurations, dashboard settings

## Status
Prototype - Business analytics and reporting.
