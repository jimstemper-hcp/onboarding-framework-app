# Account Setup Feature

## Overview
The Account Setup feature guides new pros through the initial configuration of their Housecall Pro account, including company profile, branding, and basic settings.

## Key Features
- Company profile setup
- Logo and branding upload
- Business hours configuration
- Service area definition
- Contact information setup

## Stage Contexts

### Not Attached
- Benefit messaging: "Set up your professional business profile"
- Professional image benefits

### Attached
- Onboarding items: Complete company profile, add logo
- Basic setup tasks

### Activated
- Onboarding items: Configure business hours, set service area
- Complete profile setup

### Engaged
- Profile optimization tips
- Branding best practices

## Data Dependencies
- Reads: Account data
- Writes: Company profile, branding assets, settings

## Status
Prototype - Core setup flow for new pros.
