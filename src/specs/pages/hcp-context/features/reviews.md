# Reviews Feature

## Overview
The Reviews feature helps pros collect and manage customer reviews to build their online reputation and attract new customers.

## Key Features
- Automated review requests
- Multi-platform review support
- Review monitoring
- Response management
- Review analytics

## Stage Contexts

### Not Attached
- Benefit messaging: "Build your reputation with customer reviews"
- Review impact statistics

### Attached
- Onboarding items: Enable review requests, connect review platforms
- Setup guides

### Activated
- Onboarding items: Customize request timing, set up alerts
- Advanced review features

### Engaged
- Reputation analytics
- Review response strategies

## Data Dependencies
- Reads: Job completion data, customer data, review platforms
- Writes: Review request settings, response records

## Status
Shipped - Review management available to all pros.
