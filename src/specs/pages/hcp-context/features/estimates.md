# Estimates Feature

## Overview
The Estimates feature enables pros to create professional estimates and proposals for potential customers, helping win more business.

## Key Features
- Estimate creation and customization
- Line item management
- Estimate templates
- Digital signature support
- Estimate to job conversion

## Stage Contexts

### Not Attached
- Benefit messaging: "Win more jobs with professional estimates"
- Sales conversion benefits

### Attached
- Onboarding items: Create first estimate, set up templates
- Estimate basics

### Activated
- Onboarding items: Enable digital signatures, customize branding
- Advanced estimate features

### Engaged
- Estimate analytics
- Conversion tracking

## Data Dependencies
- Reads: Customer data, pricing data, templates
- Writes: Estimate records, template customizations

## Status
Shipped - Estimates feature available to all pros.
