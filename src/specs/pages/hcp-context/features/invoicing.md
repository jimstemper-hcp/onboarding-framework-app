# Invoicing Feature

## Overview
The Invoicing feature enables pros to create and send professional invoices to their customers. It's a core billing feature that typically sees high adoption among new users.

## Key Features
- Invoice creation and customization
- Customer billing management
- Payment tracking
- Invoice templates
- Automated invoice sending

## Stage Contexts

### Not Attached
- Benefit messaging: "Get paid faster with professional invoices"
- Discovery prompts for exploring invoicing

### Attached
- Onboarding items: Create first customer, create first invoice
- Quick setup guides

### Activated
- Onboarding items: Customize invoice template, set up auto-send
- Feature optimization tips

### Engaged
- Advanced usage guidance
- Reporting and analytics

## Data Dependencies
- Reads: Customer data, job data, payment data
- Writes: Invoice records, payment records

## Status
Shipped - Core feature available to all pros.
