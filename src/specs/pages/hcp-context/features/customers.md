# Customers Feature

## Overview
The Customers feature provides CRM capabilities for managing customer relationships, contact information, job history, and communication preferences.

## Key Features
- Customer database management
- Contact information storage
- Job and payment history
- Communication preferences
- Customer tagging and segmentation

## Stage Contexts

### Not Attached
- Benefit messaging: "Keep all your customer info in one place"
- Organization benefits

### Attached
- Onboarding items: Add first customer, import existing customers
- Basic CRM setup

### Activated
- Onboarding items: Set up customer tags, configure preferences
- Advanced CRM features

### Engaged
- Customer analytics
- Segmentation strategies

## Data Dependencies
- Reads: Customer records, job history, communication logs
- Writes: Customer data, preferences, tags

## Status
Prototype - CRM functionality for customer management.
