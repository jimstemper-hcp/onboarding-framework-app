# Job Scheduling Feature

## Overview
The Job Scheduling feature provides pros with calendar management and job scheduling capabilities to organize their work and manage their team's time.

## Key Features
- Calendar view with job scheduling
- Recurring job support
- Team calendar management
- Route optimization
- Customer time slot booking

## Stage Contexts

### Not Attached
- Benefit messaging: "Keep your schedule organized and efficient"
- Scheduling efficiency benefits

### Attached
- Onboarding items: Create first job, set up work hours
- Calendar basics

### Activated
- Onboarding items: Set up recurring jobs, configure time slots
- Advanced scheduling features

### Engaged
- Team scheduling
- Route optimization usage

## Data Dependencies
- Reads: Job data, team availability, customer preferences
- Writes: Job records, schedule settings

## Status
Shipped - Core scheduling feature for all pros.
