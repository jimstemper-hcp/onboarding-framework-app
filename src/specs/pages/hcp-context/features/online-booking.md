# Online Booking Feature

## Overview
The Online Booking feature enables customers to book appointments directly through a web booking page, reducing phone call volume and capturing leads 24/7.

## Key Features
- Public booking page
- Service selection
- Time slot availability
- Instant confirmations
- Booking customization

## Stage Contexts

### Not Attached
- Benefit messaging: "Let customers book online anytime"
- Lead capture benefits

### Attached
- Onboarding items: Set up booking page, configure services
- Basic booking setup

### Activated
- Onboarding items: Customize booking flow, set availability rules
- Advanced booking features

### Engaged
- Booking analytics
- Conversion optimization

## Data Dependencies
- Reads: Schedule data, service types, availability
- Writes: Booking records, availability settings

## Status
Prototype - Online booking page functionality.
