# Service Plans Feature

## Overview
The Service Plans feature enables pros to offer recurring maintenance contracts and subscription services to customers, creating predictable revenue streams.

## Key Features
- Service plan creation
- Recurring billing setup
- Plan management dashboard
- Renewal reminders
- Plan performance tracking

## Stage Contexts

### Not Attached
- Benefit messaging: "Create recurring revenue with service plans"
- Recurring revenue benefits

### Attached
- Onboarding items: Create first service plan, set pricing
- Plan basics

### Activated
- Onboarding items: Configure renewal reminders, set up auto-billing
- Advanced plan features

### Engaged
- Plan analytics
- Retention strategies

## Data Dependencies
- Reads: Customer data, billing data, plan definitions
- Writes: Service plan records, billing schedules

## Status
Prototype - Recurring service plan management.
