# Calls Tab

## Overview
The Calls Tab manages Calendly call type definitions and booking links that reps can use to schedule calls with pros.

## Key Features
- Calendly link management
- Team-based organization
- Call duration configuration
- Description and context
- Status management

## User Stories
- As an admin, I want to add Calendly links so reps can schedule calls
- As an admin, I want to organize links by team so routing is clear
- As an admin, I want to set call durations so scheduling is accurate

## UI Components
- Call type data table
- Team filtering
- Add/Edit modal
- Duration selector
- Status controls

## Data Dependencies
- Reads: Calendly links, team assignments
- Writes: Calendly link records
