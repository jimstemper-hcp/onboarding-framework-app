# Invoicing Setup Call

## Problem
New pros who have invoicing access but haven't completed setup may need hands-on guidance to configure their invoicing workflow correctly.

## Solution
A scheduled Calendly call with the onboarding team to walk through invoicing setup, best practices, and answer questions.

## Scope
**Included:**
- Calendly booking for onboarding team
- Invoicing feature walkthrough
- Setup assistance

**Excluded:**
- Sales conversations (use sales calls)
- Technical support for existing users

## Dependencies
- **Depends on:** Pro has invoicing feature access
- **Depended on by:** Successful invoicing activation

## Success Criteria
- Pro can book a call via Calendly link
- Call covers invoicing setup steps
- Pro completes setup after call

## Functional Requirements

### FR1: Call Booking

#### User Story
As a pro who needs help with invoicing setup, I want to schedule a call with the onboarding team so that I can get personalized guidance.

#### Acceptance Criteria
| Condition | Expected Behavior |
|-----------|-------------------|
| Call type | Invoicing Setup Help |
| Team | Onboarding |
| Calendly URL | Valid booking link |
| Description | Clear explanation of call purpose |
